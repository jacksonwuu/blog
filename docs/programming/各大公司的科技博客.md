# 各种公司的科技博客

这是我收集的一份列表：各大科技公司的技术团队博客。

相比于国内科技公司来说，国外的科技公司更有技术分享的意识和意愿（除了 Amazon 和 Apple 等几个神秘的公司外）。

这些博客往往从实际应用的角度去思考技术，这些博客不仅可以作为学习材料，还可以作为了解技术最前沿应用的途径。

我个人喜欢这几个：Netflix、Google、Github、Slack
